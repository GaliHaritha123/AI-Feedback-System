AI Feedback System
This project implements an AI-powered feedback system with two modules.

Task 1
- Local prototype using Gradio
- Stores reviews in CSV
Task 2
- Production deployment using Hugging Face Spaces
- User Dashboard: https://huggingface.co/spaces/harithagali/ai-feedback-admin
- Admin Dashboard: https://huggingface.co/spaces/harithagali/ai-feedback-user

Tech Stack
- Python
- Gradio
- Pandas
- Hugging Face Spaces
